HOW TO USE THE PCBMS

1. UNZIP THE pcbms.zip file
2. MOVE THE UNZIP FILE TO xampp/htdocs/
3. Open PHPMyAdmin (http://localhost/phpmyadmin)
4. Create a database with name pcbms
5. Import pcbms.sql file(given inside the zip package in SQL file folder)
6. The Location of the pcbms.sql is pcbms/database/
7. Run the script http://localhost/pcbms
8. You will be directed to login page, the following are the login details for every user
